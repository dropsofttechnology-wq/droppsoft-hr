### 1. Appwrite Setup (10 minutes)

Follow `START_HERE.md` for step-by-step instructions:

1. Create Appwrite project
2. Create database
3. Create 7 collections (see `COLLECTIONS_SETUP.md`)
4. Enable authentication
5. Configure environment variables

### 2. Build & Deploy (5 minutes)

```bash

# Deploy to Appwrite Hosting, Vercel, or Netlify

```

### 3. Test (5 minutes)

1. Create first user
2. Login
3. Create company
4. Add employee
5. Test all features
