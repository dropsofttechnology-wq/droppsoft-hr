# ⚡ Quick Git Installation

## Step 1: Download Git (1 min)

1. Go to: **https://git-scm.com/download/win**
2. Click **"Download for Windows"**
3. The download will start automatically

## Step 2: Install Git (2 min)

1. **Run the downloaded file** (e.g., `Git-2.43.0-64-bit.exe`)
2. Click **"Next"** through all screens
3. **Keep all default options** (they're fine!)
4. Click **"Install"**
5. Wait for installation (1-2 minutes)
6. Click **"Finish"**

## Step 3: Verify Installation (30 sec)

1. **Close PowerShell completely**
2. **Open PowerShell again** (important!)
3. Run:
   ```powershell
   git --version
   ```
4. You should see: `git version 2.x.x`

✅ **If you see a version number, Git is installed!**

## Step 4: Configure Git (30 sec)

Run these commands (replace with your info):

```powershell
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

## ✅ Done!

Now you can continue with GitHub setup:
- See `QUICK_GITHUB_SETUP.md`
